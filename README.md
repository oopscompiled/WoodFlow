# WoodFlow
Автоматизация бизнес журнал листа работника лесного хозяйства для ежедневного отчёта переработанного пиломатериала (леса) в готовую пилопродукцию с дальнейшей реализацией

## Этапы разработки
1. Шаг - создать шаблон
2. Шаг - работа с данными
3. Шаг - описание логики
4. Шаг - оформление кода

## Логика работы
- Код создаёт исходный посменный файл Excel с таблицами для дальнейшего заполнения

### Назначение
Программа **автоматически генерирует ежемесячные посменные бизнес-журналы** (график 2/2) для упрощения ведения записей и исключения необходимости создавать базы вручную.

### Обработка данных
1. Создание таблиц и листов Excel по ежемесячным датам графика

### Вывод
Программа генерирует **ежемесячные посменные бизнес-журналы** (график 2/2) с самостоятельным заполнением:
- Расчёта полученного пиломатериалла
- Затрат на получение пиломатериалла
- Выходным файлом в Excel для дальнейшего редактирования и анализа
